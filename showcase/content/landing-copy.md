## Proof Strip

- Review faster

## Problem

Most automated PR summaries just retell the diff.
That sounds useful until the real review work starts: deciding what changed, what could break, what still needs confirmation, and what to test next.

## Solution

This skill reads the PR context that actually matters:
- metadata
- diff
- review comments
- check status

Then it outputs a compact checklist instead of a wall of text.
